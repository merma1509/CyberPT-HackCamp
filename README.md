# CyberPT-HackCamp: Positive Technologies HackCamp 2025

> A curated archive of labs, techniques, writeups, and personal insights gathered during the 2025 Positive HackCamp held between **July 26 - August 10**.

## Overview

This repository documents my journey and learning during Positive HackCamp — an intensive international cybersecurity bootcamp hosted by Positive Technologies. The camp covered practical, real-world scenarios in Web Exploitation, Network Attacks, Reverse Engineering, Forensics, Privilege Escalation, and more.

## Structure

- `01-Introduction`: High-level summary of the camp, goals, and personal reflections.
- `02-Tools-Used`: Breakdown of major tools and utilities used across labs.
- `03-Solved-Labs`: Step-by-step solutions to the labs and challenges I worked on.
- `04-Writeups`: Detailed writeups using a consistent format.
- `05-Techniques`: Reusable attack techniques and exploitation strategies.
- `06-Resources`: Helpful external links, reading list, cheat sheets.
- `07-Scripts-and-Tools`: Scripts I wrote or used to automate tasks or check things like VPN setup, email recon, etc.
